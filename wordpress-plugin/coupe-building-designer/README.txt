A 3D Building Designer with floor plans and real-time pricing.

== Description ==

This plugin adds a "Coupe Building Designer" to your WordPress site.
It features:
* Real-time 3D Visualization using Three.js
* Interactive Floor Plan using GoJS
* Instant Pricing Calculation
* Quote Submission via Email

== Installation ==

1. Zip the `coupe-building-designer` folder.
2. Go to your WordPress Admin > Plugins > Add New > Upload Plugin.
3. Upload the zip file and click "Install Now".
4. Activate the plugin.
5. Create a new page (or edit an existing one) and add the shortcode `[coupe_designer]`.

== Configuration ==

* To change pricing, edit `assets/js/pricing.js`.
* To change colors, edit the options in `templates/interface.php` or `assets/js/building-3d.js`.
